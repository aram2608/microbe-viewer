# microbe-viewer
---

An over-engineered website created for the Microbial Evolution and Ecology course at UNH.
A simple educational website that covers a wide range of microbial ecology topics, ranging from extremophiles, symbionts, and parasitic organisms.

The tech stack is way over burdensome for the small scope of the project, built using `fastAPI` and all its dependencies.

This could have been a static site but I like Python and wanted to play around with `fastAPI`.

Enjoy.
